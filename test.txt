Git commands:
$pwd (Print Window Directory)
$mkdir - Make a new folder
$cd - Get into a folder
$cd .. Get out of the folder
$ls or dir -  list all the documents in the curent directory
$git init - initializes a new repo 
$git add . add the codes to be commited
$git commit -m "" - staging codes ready to be pushed.
$touch name.extension (Creates a file with the respective name and doc type in the active directory)
$rm filename.extension- removes/deletes a file in the current directory.
$git config --global user.name "<your name>"
$git config --global user.email "<your email>"
